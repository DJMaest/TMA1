In this unit we will discuss the fundamentals of javascript as a client side side scripting language.

Javscript

Introduction

Javascript is essentially the standard clientside scripting language. So far we have learned about html and css. These technologies allow us to render visually stunning layouts and pages. However, our pages still miss a key ingredient. They are all static and lack the dynamic and responsive side. Our html and css pages essentially can't "do something". The solution to making our pages dynamic and have the ability to handle various inputs and events is Javascript. Javascript allows are pages to come to life and be responsive in the sense of handling events and responding to inputs.

Features of Javascript

  • Window object: Window object essentially gives us access to the browser window. We can use this to reload a browser window or detect when a window has loaded using "window.onload()". We can also display alert messages on the browser using for exampe, "window.alert("This is an alert!");".
  • DOM Manipulation: DOM stands for Document Object Model. Using javascript you can access the document object. The document object is an abstraction of a corrosponding rendered html document. using dockument we can print out text using "document.writeline("Some String");" for example which prints out the string to the body of the document. We can also use dom objects in order to dynamically change their style. For instance, "domObject.style.color = 'green'" is an illustration of what can be done.
  • Case sensitive: Variables and methods in javascript are case sensitive. This is common in other programming languages as well.
  • Mathematical operations: Javascript supports multiple arithmetic operations such as +,-,/,*; Other operations such as rounding down can also be performed using the Math object. For example "Math.floor(1.43);".
  • Case sensitive: Variables and methods in javascript are case sensitive. This is common in other programming languages as well.
  • String manipulation: Javascript supports string concatentaion and parsing. For instance, you can concatenate two strings using the '+' operand. e.g. 'a' + 'b' will evaluate to 'ab'.
  • Logical operations: Javascript supports essentially all logical operators, encompassing, OR (||), AND (&&), equal (==), not equal (!=), greater than (>), less than (<), etc.
  • Loops: Javascript supports both while and for loops
  • Implicit types: In Javascript data types are determined implicitely and we mainly worry about variable scope using "var" and "let"; and variable immutability using "const".

Note: Javascript has two types of equal operator. Strict equal (===) and equal (==). Strict equal is closer to what is seen other programming languages. Using normal equal (==) in javascript should be considered with caution. For example "12" == 12 returns true even though one is a string and the other is number. Note: Semi-columns are not necessary in JavaScript.

Handling Events

There are two way to approach this. We can use inline event handlers or external event handlers that are referenced using script tags. Inline events can be referenced using attributes such as "onclick="eventHandler()"". Here "eventHandler()" is the name of the javascript function that is called when the element is clicked.
e.g. <button onclick="handleClick()" > Some button </ button> calls handleClick(), where it can be defined in a script tag or within a javascript file.
function handleClick() {  window.alert("Clicked!"); } For external event handlers, we need to use DOM extraction techniques to store DOM objects in javascript object and attach an event handler to these objects. For example, if our button has an id attribute, then we can use this id attribute to reference this element and store it in a javascript object using a document query.
e.g. <button id="specialButton" > Some button </ button>
We can handle click events with the following javascript snippet:
const buttonObject = document.getElementById("specialButton"); buttonObject.addEventListener("click", ()=> {  window.alert("Clicked!"); });
